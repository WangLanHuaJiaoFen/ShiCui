# ShiCui

ShiCui 是一款简洁的句子收藏APP，使用[一言API](https://developer.hitokoto.cn/sentence/)，目前有Android和Windows版本。

    ![句子分类收藏](https://s2.loli.net/2024/09/23/7EbDcWTJ43vXe2r.jpg)
